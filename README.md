# 🎬 WatchScape – Social Movie Tracking Platform

A full-stack social movie application that allows users to track watched films, manage watchlists, and interact with others through posts and social features.

🌐 Live: https://watchscape.vercel.app  
💻 Backend: https://github.com/pasindunavodya01/watchscape_server  

---

## ✨ Features

### 🎥 Movie Tracking
- Add movies to **Watched** or **Watchlist**
- Automatically generate posts when adding movies
- View detailed movie information with posters and metadata

### 👥 Social Features
- Create and share posts
- Like, comment, and interact with others
- Follow/unfollow users
- Notifications system
- Pin favorite movies to profile

### 🔍 Discovery & Search
- Search movies using IMDB API
- Explore user profiles and activity
- View detailed movie pages

### 👤 User Profiles
- Custom user profiles
- Display watched movies and collections
- Social activity tracking

---

## 🛠 Tech Stack

- **Frontend:** React  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **API Integration:** IMDB API  
- **Architecture:** REST APIs  

---

## 🚀 How It Works

- Users can search and add movies using IMDB data  
- Adding a movie automatically creates a social post  
- Users interact through likes, comments, and follows  
- Backend handles API requests and database operations via RESTful services
